General
Core extension behavior
HPP Active
When disabled, the extension will not intercept login forms or show the Presence Gate on any site.
Certificate Lifetime
How long a Presence Certificate is valid for session reuse. The server may issue a shorter lifetime than this setting if its policy requires it.
Privacy
Data handling and logging
Privacy Mode
Disables all diagnostic logging and telemetry. Certificate metadata is still stored in session memory for session management.
Debug Logging
Writes HPP protocol events to the browser console. Sensitive fields are redacted. Disable in production use.

Advanced
Development and debugging options
Server Endpoint Override
Override the HPP Attestation Server URL. Leave blank to use the production server. Set to http://127.0.0.1:9443 to use the local mock server during development.
Max Clock Skew (ms)
Maximum tolerated difference between device clock and server clock. Range: 10000–60000. Default: 30000. Overridden by server configuration when available.
